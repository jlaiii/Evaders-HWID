Key Features

Designed for robustness and real-time system introspection.

System HWID Collection

Accurately collects and stores hardware identifiers (HWIDs) for system components, providing a detailed baseline.

Real-time Comparison

Compares the current HWID signature against saved baselines to detect and track system changes.

Comprehensive Statistics

View detailed statistics on HWID changes, collection timestamps, and system monitoring history.

Anti-Cheat Simulation

Simulates anti-cheat system monitoring to test system resilience and detect potential HWID tracking or validation issues.

Configurable Settings

Manage application settings, including monitoring intervals, startup behavior, and data file locations.

Detailed Event Logging

Every action, change, and system check is logged for complete auditability and troubleshooting.

Getting Started

Evaders HWID is a Python utility. Follow these steps to set up and run the tool.

1. Prerequisites

Ensure you have Python 3 installed on your system.

2. Download & Setup

Clone the repository or download the source code:

git clone https://github.com/jlaiii/Evaders-HWID.git
cd Evaders-HWID

3. Install Dependencies

The tool requires `psutil`. Install it using pip:

pip install -r requirements.txt

*(Note: The tool attempts to auto-install dependencies if run directly.)*

4. Run the Tool

Execute the main script from your terminal:

python evaders_hwid.py
